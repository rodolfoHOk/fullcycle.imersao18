## Tecnologias

- Typescript / Javascript
- Node.js
- Nest.js
- Prisma.js
- MySQL
- Docker

## Nest.js

- npm install -g @nestjs/cli
- nest new project-name
- nest g resource
- nest g module module-name
- nest g service service-name

## Prisma.js

- npx prisma init
- npm install @prisma/client
- npx prisma migrate dev
- npx prisma studio
- npx prisma generate
- npx prisma migrate reset
